# 《系统设计说明书》模板

仅在生成、补全或改写《系统设计说明书》时读取并使用本模板。根据项目规模合并不适用的小节；删除占位说明前，确认相关内容已经完成或明确说明“不涉及”及理由。

## 使用规则

- 先读取立项书、需求说明书和已有设计材料，不凭模板补造内容。
- 用项目真实名称替换占位符，未知内容写为 `【待学生确认：问题】`。
- 让架构、页面、接口、状态和数据采用统一业务术语。
- 优先描述核心业务闭环，次要功能保持简洁。
- 图表应有编号、标题和正文引用；附件应有明确路径或名称。
