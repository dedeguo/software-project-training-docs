# 项目选题库

| 难度 | 项目 | 核心业务 |
|---|---|---|
| L1 | 校园失物招领系统 | 发布、认领、审核、归还 |
| L1 | 实验室预约系统 | 时段发布、预约、审核、签到 |
| L2 | 学生竞赛管理系统 | 赛事、组队、报名、审核、成果 |
| L2 | 毕业设计过程管理系统 | 选题、双选、任务、检查、答辩 |
| L2 | 校园活动管理平台 | 发布、报名、签到、评价、统计 |
| L3 | 课程学习支持平台 | 资源、任务、学习记录、评价 |
| L3 | 景区智能导览系统 | 景点、路线、问答、推荐 |
| L4 | 课程知识库问答系统 | 文档、检索、问答、反馈 |
| L5 | 个性化学习多智能体系统 | 画像、协商、资源生成、评价 |

学生可以自选题，但必须通过范围和可行性检查。
