# AI辅助编程工作流

## 从一个差提示词开始

不推荐：

```text
帮我写一个学生管理系统。
```

推荐：

```text
请先分析当前Spring Boot项目结构，不要立即修改代码。

任务：实现学生分页查询。
环境：JDK 17、Spring Boot 3、MyBatis、MySQL 8。
现有规范：使用Result<T>统一返回，全局异常处理已经存在。

验收标准：
1. GET /api/students；
2. 支持page、size、keyword；
3. keyword匹配姓名或学号；
4. 校验分页参数；
5. 不修改无关模块；
6. 先给出修改计划和涉及文件；
7. 完成后给出Apifox验证步骤。
```

## 一个完整提示词应包含

- 项目背景；
- 当前技术栈；
- 已有代码和规范；
- 本次具体任务；
- 不能做什么；
- 可验证的验收标准；
- 希望AI采用的工作步骤。

## 实践任务

选择示例项目中的一个小功能，完成：

1. 让AI阅读项目结构；
2. 让AI提出修改计划；
3. 审核计划后分步骤实现；
4. 编译并运行；
5. 使用Apifox测试；
6. 提交Git记录；
7. 填写AI协作记录。
